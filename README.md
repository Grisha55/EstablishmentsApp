# Приложение EstablishmentsApp (для поиска ресторанов)

## Какие технологии были использованы в проекте:
- Верстка UI с помощью кода
- GCD
- MVP
- Firebase (регистрация по номеру телефона)
- Кастомные UI элементы
- GoogleMaps
- Создал NetworkLayer
